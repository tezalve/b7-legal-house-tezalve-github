Difference between Flex-box & grid?

Flexbox was created for layout in one dimension, which can be either a row or a column, as opposed to CSS Grid Layout, which was developed for layout in two dimensions. Grid was intended for two-dimensional layout with simultaneous rows and columns.

What is the purpose of media query?

We can use media queries to apply CSS styles based on the basic kind of a device (such as print vs. screen) or other details like screen resolution or browser viewport width.

What is css box model & how does it work?

The phrase "box model" is used in CSS when discussing design and layout. Every HTML element is essentially enclosed in a box thanks to the CSS box model. It is made up of margins, borders, padding, and the content itself.

What is semantic tag?

Search engines and other user devices can better understand the significance and context of online pages thanks to semantic HTML elements. Semantic components make web pages considerably easier to read. It is more easily accessible. Better user experience is provided.
